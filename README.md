# Data-Visualization-with-Python---IBM-

Analysis of Historical Automobile Sales Data. This repository contains the final project for the IBM Data Visualization with Python course. It includes:

- Exploratory Data Analysis (EDA) Notebooks: Visualizations (Line, Bar, Scatter, Pie, Bubble plots) created using Matplotlib and Seaborn to analyze sales trends, seasonality, and recession impacts (Tasks 1.1 - 1.9).

  - Full EDA Code; Exploratory Data Analysis.ipynb
  - All plots in the plots folder.

- Interactive Dashboard: A complete, dynamic web application built with Dash and Plotly Express for live filtering and reporting of both Recession Period and Yearly statistics (Task 2.1 - 2.6).

  - Full Code; Interactive Dashboard.py
  - Screenshots of the Dashboard in the dash folder.
